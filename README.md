# Usage
<!-- usage -->
```sh-session
$ npm install -g whatsapp-explorer
$ whatsapp COMMAND
running command...
$ whatsapp (-v|--version|version)
whatsapp-explorer/0.0.0 darwin-x64 node-v12.16.1
$ whatsapp --help [COMMAND]
USAGE
  $ whatsapp COMMAND
...
```
